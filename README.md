# Validation-VR-Samuel

# Plage astrale

Ce projet de validation interne vise à créer un univers en réalité virtuelle sur A-frame. Mon projet a pour base une île entouré d'un océan où le joueur peut utiliser des pierres pour bouger une lune artificielle. Deux animations sont possible grâce aux pierres et une dernière est possible grâce au poulpe qui est caché dans l'océan. La lune s'abbat sur le poulpe afin de le terrasser. Ce projet a pour objectif d'utiliser premièrement le nouveau componenet a-ocean et d'utiliser divers modèles GLTF pour créer un environement crédible. Le deuxième est d'utiliser le principe d'animation et de bouger des objets en en regardant d'autres.
